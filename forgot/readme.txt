****************************Cara Penggunaan*************************************
1. Edit File "config.php" menggunakan notepad
2. buat sebuah database baru dengan nama "latihan_forgot"
3. kemudian import file "latihan_forgot.sql" ke dalam database yang tadi anda
   buat melalui phpmyadmin
4. lalu setting/ubah nama $host, $user, $pass, $db, dan $my_link yang ada di 
   dalam file tersebut sesuai degan settingan database anda yang tadi anda buat
3. Save. Dan jalankan
